# Tourism Management System

I built this project to manage travel bookings and customer data through a simple web interface. 
Users can register, log in, browse destinations, and make reservations — all stored in a database.

## Why I built this

Managing tour bookings manually is messy. This project tries to make that process cleaner 
by giving users a simple interface to handle everything in one place.

## Tech used

- Python + Flask for the backend
- HTML and CSS for the frontend
- SQLite for the database

## How to run it

Make sure you have Python installed, then:

```bash
git clone https://github.com/naveenputta68/TOURISM-MANAGEMENT-SYSTEM.git
cd TOURISM-MANAGEMENT-SYSTEM
pip install -r requirements.txt
python app.py
```

Then open your browser and go to `http://127.0.0.1:5000`

No extra database setup needed — tourism.db is already included.

## What it does

- Users can register and log in
- Browse available travel destinations
- Book a trip and view their reservations
- All data is saved and retrieved from the database
